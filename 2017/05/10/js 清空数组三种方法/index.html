js清空数组三种方法

发表于 2017-05-10 |

方法一：splice

定义及用法

splice()方法向数组中添加/删除元素，返回被删除的项目。
注释：该方法会改变原是数组

语法

arrayObject.splice(index,howmany,item1,item2...itemx)

参数 描述
index 必须。整数，规定添加/删除项目的位置，使用负数可以从数组结尾处规定位置。
howmany 必须。要删除的项目数量。如果设置为0，则不会删除。
item1…itemx 可选。向数组添加的新项目。
返回值

Array类型，返回被删除的元素。

说明

splice() 方法可删除从 index 处开始的零个或多个元素，并且用参数列表中声明的一个或多个值来替换那些被删除的元素。

如果从 arrayObject 中删除了元素，则返回的是含有被删除的元素的数组。
